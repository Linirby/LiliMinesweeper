# Minesweeper
### For Linux
To execute, type `./lili_minesweeper` in shell
(if no perms type `chmod +x lili_minesweeper` and retry)
### For Windows
To execute, type `lili_minesweeper.exe` in cmd or powershell
### Controls
- `ARROWS` to change selected case
- `D` to discorver (open) a case
- `F` to set a flag on a case
- `ESCAPE` to quit
